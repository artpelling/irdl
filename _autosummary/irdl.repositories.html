classmethod initialize(doi, archive_url)#

Initialize the data repository if the given URL points to a corresponding repository.

Initializes a data repository object. This is done as part of a chain of responsibility. If the class cannot handle the given repository URL, it returns None. Otherwise a DSpaceRepository instance is returned.

Parameters:
doistr

The DOI that identifies the repository.

archive_urlstr

The resolved URL for the DOI.
